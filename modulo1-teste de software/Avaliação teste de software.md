# 📘 Avaliação – Fundamentos de Teste de Software

### 🧪 ***O que é teste de software?***
**R:** É o processo de conferir se um sistema está funcionando corretamente, com o objetivo de encontrar erros antes que o usuário o utilize.

### 💼 ***Por que testar é importante para o negócio?***
**R:** Ajuda a encontrar erros antes que o sistema chegue ao cliente. Isso evita problemas, reduz gastos e melhora a confiança no produto.

### ⭐ ***O que caracteriza um software com qualidade?***
**R:** É um software que atende ao que o usuário precisa, é fácil de usar, seguro e rápido.

### 👨‍💻 ***Qual o papel do QA em um time moderno?***
**R:** Garantir que o software tenha qualidade. O QA ajuda a encontrar erros e trabalha com a equipe para entregar um software mais seguro, confiável e com menos problemas.
